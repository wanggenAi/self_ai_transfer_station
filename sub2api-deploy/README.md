# Sub2API Local Deploy Bundle

This folder is prepared for local-only deployment on macOS/Linux using Docker Desktop + Docker Compose.

## Security defaults
- `BIND_HOST=127.0.0.1` (localhost only)
- Strong random secrets are generated by script
- `.env` and runtime data are git-ignored

## Quick start
1. Initialize env and secrets:
```bash
cd sub2api-deploy
./scripts/init-env.sh
```

2. Start services:
```bash
./scripts/up.sh
```

3. Check status:
```bash
./scripts/status.sh
```

4. Open Web UI:
- http://localhost:8080

5. Get initial admin credentials from logs:
```bash
./scripts/admin-creds.sh
```

## Daily operations
- Follow logs:
```bash
./scripts/logs.sh
```
- Follow one service logs:
```bash
./scripts/logs.sh sub2api
```
- Stop services:
```bash
./scripts/down.sh
```

## Roo Code settings
- API Provider: OpenAI Compatible
- Base URL: `http://localhost:8080/v1`
- API Key: generated in Sub2API admin panel
- Model: use exact model name shown in Sub2API
