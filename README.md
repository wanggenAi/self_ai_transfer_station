# self_ai_transfer_station

This repo now includes a ready-to-run local deployment bundle for `Wei-Shaw/sub2api`.

## Use this folder
- `sub2api-deploy/`

## One-command flow
```bash
cd sub2api-deploy
./scripts/init-env.sh
./scripts/up.sh
```

Then open:
- http://localhost:8080
